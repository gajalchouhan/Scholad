This is the simple Data-table.
1.) Search :-
  Search is working based on global search.
2.) Sorting :- 
  When you will click the header it will sort first in asending order then desending order after it come as default.
3.) Selecting :-
  Option is 'position' and 'office'. Based on selection you can see the column.
4.) Table Row:-
  For more row details you click the table row. Details will pop up. after again click it will hide.
5.) Pagination :-
Each page has 10 rows. You can select any page. Using previous or next arrow click button you can select any page.
